# Extension Icons

This directory should contain the extension icons required for Chrome Web Store submission.

## Required Icon Sizes

Create PNG icons with the following dimensions:

1. **icon16.png** - 16x16 pixels
   - Used in the extension management page

2. **icon48.png** - 48x48 pixels
   - Used in the extensions management page and toolbar

3. **icon128.png** - 128x128 pixels
   - Used in Chrome Web Store
   - Actual icon should be 96x96 with 16px transparent padding on all sides

## Design Guidelines

- **Format**: PNG only
- **Style**: Simple, recognizable design
- **Color**: Should work on both light and dark backgrounds
- **Perspective**: Face viewer directly, avoid dramatic angles
- **Brand**: Consistent with "Momo" brand (cute peach theme üçë)
